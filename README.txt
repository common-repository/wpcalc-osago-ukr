Калькулятор ОСАГО по тарифам Украины

== Description ==

Плагин калькулятора ОСАГО для CMS WordPress с тарифами Украины. Простая интеграция плагина и настройка коэффициентов под Вас. Калькулятор рассчитывает базовый страховой платеж по ОСАГО, а также страховой платеж со скидкой. После чего клиент имеет возможность отправить запрос в страховую компанию или страховому агенту на почту, указанную в настройках калькулятора.

== Installation ==

1. Загрузите `wpcalc-osago-ukr` папку в директорию  `/wp-content/plugins/`.
2. Активируйте плагина в разделе "Плагины" в админпанели WordPress.
3. Перейдите на страницу настроек (OSAGO UKR) - в разделе "Настройки" в админпанели Вашего сайта.

Для вставки на страницу или в запись используйте шорткод [wpcosagoukr]
